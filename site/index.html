Flashcard Extension + Gesture AI

Chrome extension with hand-gesture difficulty grading & spaced repetition.

Gesture DemoSource Code

Features

Flashcards

Create from highlighted text or context menu.

Gestures

Thumbs up/down, peace, pinch, OK, fist, point, palm.

SRS Engine

Intervals, ease factor, repetitions, next review scheduling.

Analytics

Difficulty counts & mastery progress.

How to Use the Chrome Extension

  1. Clone repository
  2. Run npm install && npm run build
  3. Open chrome://extensions → Enable Developer Mode
  4. Load unpacked → select extension folder
  5. Highlight text on any page → open popup → Add Highlighted Text

Local Development

npm install
npm run dev:site
# open http://localhost:5500

Gesture Demo

Open the gesture test page and allow camera access. Perform gestures to see mapped difficulty labels.

Deploy to Netlify

  1. Connected repository in Netlify
  2. Build command: npm run build:netlify
  3. Publish directory: site
  4. Deploy
